# Pricing

Anyone who wants to self-host can use the TNRA app for free. It's open-source and you can access the code
[here](https://github.com/dogeared/tnra)
:: Hosted per Member
**$5** / month / member

- All data is encrypted at rest
- Optional Slack integration
- Manage the monthly call chain
- Members can export their own data at any time
- Chat support with 48-hour response time

[Request Access](/?to=request-access)
:: Hosted Group {featured}
**$25** / month

- Everything in Hosted Individual
- max 8 members
- Chat support with 24-hour response time


[Request Access](/?to=request-access)
:: Self Hosted
**$0**

- open source and free to use on GitHub
- All data is encrypted at rest
- Runs on the cloud or local service of your choice
- Supports Cloudflare tunnels
- Build and install docs available
